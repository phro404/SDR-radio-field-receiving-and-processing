# SDR-radio-field-receiving-and-processing
  With the help of a SDR ModeS and ModeA/C surveillance data shall be received and processed.

# Tools:
  dump1090: https://github.com/flightaware/dump1090
  see also: **tools** directory
  In here you have all the tools needed to build dump1090-

# SRC:
  This folder contains all necessary Python-scripts. There is already a dump1090 file included, which has been compiled with Ubuntu 20.04. In case the program does not work         properly, dump1090 should be compiled again (further information in the readme-file in the tools-directory). 
  Execute gui.pyw with Python3 to start the program, the SDR has to be connected.
